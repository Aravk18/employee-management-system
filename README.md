# 🧑‍💼 Employee Management System

A modern Employee Management System built using **ReactJS, TailwindCSS, and JavaScript**.  
This system allows Admins to assign tasks to employees and employees to track and update their task progress.

---

## 🚀 Features

### 🔐 Authentication
- Separate login for Admin and Employees
- Role-based dashboard access

### 👨‍💼 Admin Dashboard
- Create and assign tasks to employees
- View task statistics
- Track:
  - Active Tasks
  - Completed Tasks
  - Failed Tasks
  - New Tasks

### 👨‍💻 Employee Dashboard
- View assigned tasks
- Update task status
- Track task progress
- Clean and responsive UI

---

## 🛠 Tech Stack

- ⚛️ ReactJS
- 🎨 TailwindCSS
- 🟨 JavaScript
- 💾 LocalStorage (as database)
- ⚡ Vite
